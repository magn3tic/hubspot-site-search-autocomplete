About The Module

The Hubspot Autocomplete Site Search is a "global module", this means you will adjust the options at the template level. Then anywhere the module is placed in a template, it will inherit these global settings

Module Options

  • Portal Id - defaults to the current portal ID you are using, can be overridden by entering any other Hubspot Portal ID.
  • Accent Color - adds a colored underline to the search input when focused, and for the included svg vector icons.
  • Label Text - enter text for the label element above the input, you can also disable the label all-together.
  • Input Placeholder - set a "placeholder" for the input (the text that shows in the input before the user begins typing).
